# 📘 Bit Notes Sharing System

## 📌 About Project
A full-stack web application where students can upload, share, and access notes easily.  
This system helps students collaborate and manage study materials efficiently.

---

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 📤 Upload Notes
- 📄 View Notes Dashboard
- 🧑‍🎓 Student-friendly interface
- 📂 Organized notes management

---

## 🛠 Tech Stack
- Java (Spring Boot)
- MySQL (Database)
- HTML
- CSS
- JavaScript
- Maven

## 🗄 Database Setup

- MySQL database used
- Configure database in:
src/main/resources/application.properties

Example:
spring.datasource.url=jdbc:mysql://localhost:3306/your_db_name
spring.datasource.username=root
spring.datasource.password=your_password
